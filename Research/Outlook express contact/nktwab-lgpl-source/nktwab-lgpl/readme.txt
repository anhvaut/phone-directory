Compiling
=========

* NKTWab Library
Visual Studio project files can be found in: src/lib
There are two project files you can use to compile the libray:
  * nktwab.sln : Visual Studio 2003
  * nktwab.dsw : Visual Basic 6.0

  
Requirements
============


* This NKTWab release requieres Comet 1 beta 31 to compile.
  Comet can be found in: http://www.lambdasoft.dk/comet/index.htm


Directories List
================

* src/lib : NKTWab Library Source Code
* src/demo : NKTWab Demos
* src/demo/dotNet : VB .NET NKTWAB Demo project
* src/demo/vb6 : Visual Basic 6.0 NKTWAB Demo project
